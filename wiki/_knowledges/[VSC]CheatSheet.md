## SHORTCUTS COMMANDS

  GENERAL :  
    Keypress Command  
    `Ctrl + Shift + P`   Command pallette  
    `Ctrl + Shift + /`   Toggle comment

  FILE NAVIGATION :  
    `Ctrl + P`                 Goto Anything  

  LINE MANIPULATION :  
    `Ctrl + ], [`     Indent/outdent  


  WORD MANIPULATION :  
    `Ctrl + Delete/Backspace`   Delete word forward/backward  


  SELECTION :  
    `Ctrl + D`              Quick select (repeat to add next)
